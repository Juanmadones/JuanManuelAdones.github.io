Octomanía

  • •
  • •
  • •

This project was made with Unreal Engine , I wanted to create some minigames to get used to Unreal Engine and see how to publish it in the PlayStore, making the game was "the easy part", the concept is to hold what you can and get the maximum score possible. The player directs a ball that changes color every time it goes from octagon to octagon, this octagons shrink until they kill the player if this is not fast enough, as the game goes on, colors are added to complete the 8 faces. There are 3 powerups to help the player and one that gives points. As you can imagine it is not suitable for color-blind people but it ended up being a fun minigame.

Undoubtedly the most problematic was to make it work perfectly once published, Unreal had conflicts with the version of the billing library that uses the PlayStore, luckily it was fixed soon after. I used Android studio for testing and configuration.

With this game I learned how to register a game, how to develop it from 0 to 100 in Unreal and how to publish it on PlayStore.
